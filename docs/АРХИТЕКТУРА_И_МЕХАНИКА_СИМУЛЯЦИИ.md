```python
        # Fallback: 10% —à–∞–Ω—Å —Å–ª—É—á–∞–π–Ω–æ–≥–æ –¥–µ–π—Å—Ç–≤–∏—è –ø—Ä–∏ –Ω—É–ª–µ–≤—ã—Ö –≤–µ—Å–∞—Ö
        if random.random() < 0.1:
            return random.choice(['Post', 'Purchase_L1', 'SelfDev'])
        return None
        
    return random.choices([n for n, _ in actions], weights=weights)[0]
```
